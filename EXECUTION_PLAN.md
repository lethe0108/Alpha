## 🎯 关键决策点 (需要用户确认)

### D1: 技术选型确认 ⏳

**建议方案**:
| 层次 | 技术选择 | 理由 |
|------|---------|------|
| 后端 | FastAPI + SQLAlchemy | 高性能、自动文档、易维护 |
| 前端 | Vue 3 + TypeScript | 类型安全、组件化、生态好 |
| UI 库 | Element Plus | 组件丰富、文档完善 |
| 图表 | ECharts | 功能强大、交互性好 |
| 数据库 | SQLite | 轻量、无需配置、易备份 |

**请确认**: 回复"确认"或提出修改意见

---

### D2: 数据源选择 ⏳

**可选方案**:
| 数据源 | 类型 | 费用 | 说明 |
|--------|------|------|------|
| **iTick** | 基本面 + 技术面 | 付费 | 数据全面、API 稳定 |
| **QVeris** | 资金流 | 付费 | 主力资金数据 |
| **AllTick** | 实时监控 | 付费 | 实时行情、新闻 |
| **Yahoo Finance** | 基础数据 | 免费 | 适合开发测试 |
| **AKShare** | A 股数据 | 免费 | 开源、数据全 |

**建议**: 开发阶段使用免费数据源 (AKShare/Yahoo)，生产环境切换付费源

**请确认**: 回复"确认"或指定数据源

---

### D3: 部署方式 ⏳

**可选方案**:
| 方案 | 优点 | 缺点 | 适用场景 |
|------|------|------|---------|
| **本地部署** | 快速、免费、数据本地 | 仅限本机访问 | 个人使用 (推荐) |
| **Docker** | 可移植、易备份 | 需要学习成本 | 多设备同步 |
| **云服务器** | 随时访问、专业 | 需要费用、运维 | 商业版 |

**建议**: MVP 阶段本地部署，后续考虑 Docker 封装

**请确认**: 回复"确认"或指定部署方式

## 🤖 Agent 协作机制

### 任务分配

```python
# 总指挥自主拆解任务
任务列表 = [
    {
        "id": "1.1",
        "name": "项目结构搭建",
        "agent": "后端",
        "priority": "P0",
        "estimated_hours": 2,
        "deadline": "2026-03-18 14:00"
    },
    # ... 更多任务
]
```

### 进度汇报 (每 20 分钟)

每个 Agent 自动更新 `PROJECT_TRACKING.md`:

```markdown
## 📬 Agent 汇报记录

### 后端开发工程师 - 11:45 汇报
**状态**: 🟢 进行中
**当前任务**: 项目结构搭建
**已完成**:
- [x] 创建 backend/目录结构
- [x] 配置 FastAPI 基础
**进行中**:
- [🔄] 数据库模型设计 (50%)
**问题**: 无
**下一步**: 完成 models 定义
```

### 问题上报

```
当遇到阻塞问题时:

1. Agent 立即更新 PROJECT_TRACKING.md 的"🚨 风险与问题"部分
2. 总指挥收到通知后评估:
   - 技术难点 → 分配其他 Agent 协助
   - 方向问题 → 请求用户确认
   - 资源问题 → 调整计划
3. 问题解决后继续执行
```
