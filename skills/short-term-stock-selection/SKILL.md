## 🎯 核心分析框架

### 五维分析模型

```
                    短线选股
                        │
        ┌───────┬───────┼───────┬───────┐
        │       │       │       │       │
    政策面   消息面   技术面   资金面   情绪面
        │       │       │       │       │

| 维度 | 权重 | 分析内容 | 数据来源 |

## 📋 每日分析流程

### 盘前分析（9:00-9:15）

```
第 1 步：政策面扫描
├── 搜索"国务院/证监会/央行 最新政策"
├── 搜索"十五五规划 最新进展"
├── 搜索"行业政策 利好/利空"
└── 记录政策受益板块

第 2 步：消息面扫描
├── 搜索"上市公司 公告 重大合同/业绩预增"
├── 搜索"行业 新闻 利好"
├── 搜索"地缘政治/国际形势 影响"
└── 记录消息催化个股

第 3 步：外盘分析
├── 美股隔夜表现

└── 汇率变化

第 4 步：生成盘前策略
├── 热点板块预测
├── 关注个股列表
├── 仓位建议
└── 风险提示
```

```
实时监控:
