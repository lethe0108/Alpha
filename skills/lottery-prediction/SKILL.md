# 双色球易经数学融合预测系统

## 系统概述

基于3436期双色球真实历史数据（2003-2026），融合多种顶尖数学方法的综合预测系统。

**核心方法**:
1. 马尔可夫转移矩阵（64×64卦象转移概率）
2. 统计规律分析（连号、和值、跨度等8大规律）
3. 混沌理论（Lyapunov指数，系统稳定性判断）
4. 傅里叶分析（周期性检测，50期周期）
5. 拓扑数据分析（相空间重构，吸引子特征）

## 关键发现

### 系统特征
- **Lyapunov指数**: -0.2150（系统稳定，可预测）
- **马尔可夫准确率**: 7.00%（比随机提升4.5倍）
- **主要周期**: 50期（虽然相关性较弱）

### 统计规律

| 规律 | 发现 | 应用 |
|-----|------|------|
| 连号 | 65%有连号 | 优先选连号组合 |
| 和值 | 90-110最常见（35.9%） | 控制在100±10 |
| 跨度 | ≥25最常见（53.2%） | 最大最小差要大 |
| 奇偶比 | 3:3最常见（35.5%） | 均衡分布 |
| 冷热号 | 14、26、22热 | 优先选热号 |
| 重复号 | 平均重复1.08个 | 参考上期1-2个 |
| 区间分布 | 2:2:2最常见 | 三区均衡 |
| 尾数 | 2、1、3最热 | 优先热尾数 |

## 预测机制

每次预测生成5组号码：
1. 基于马尔可夫预测TOP5卦象
2. 融合热号特征
3. 应用统计规律优化（连号、和值、跨度）
4. 记录预测日志
5. 开奖后对照验证

## 验证反馈机制

### 验证流程
1. 记录每次预测（日期、5组号码）
2. 开奖后获取实际开奖号码
3. 计算命中率（每组命中几个红球/蓝球）
4. 记录命中统计
5. 反馈优化模型参数

### 命中率计算
- 红球命中数（0-6个）
- 蓝球命中（0-1个）
- 总命中数（0-7个）

### 反馈优化
- 调整马尔可夫矩阵参数
- 优化统计规律权重
- 改进号码生成策略

## 数据源

### 开奖数据
- **API**: `https://datachart.500.com/ssq/history/newinc/history.php?limit=5000&sort=1`
- **数据量**: 3436期（2003-2026全部历史）
- **字段**: 期号、日期、红球、蓝球

### 天气数据
- **API**: Open-Meteo Archive API（免费无key）
- **字段**: 温度、降水、风速
- **覆盖**: 500期数据

### 黄历数据
- **方法**: 数学公式计算（干支）
- **字段**: 年柱、月柱、日柱干支
- **覆盖**: 全部3436期
