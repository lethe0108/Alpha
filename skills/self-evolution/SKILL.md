# Self-Evolution Skill

## Description
自我进化技能，用于 AI 助手的持续学习和能力提升。利用外网访问优势，学习最新技术，增强现有能力，创建新技能。

## Activation
当用户提到"自我进化"、"能力提升"、"学习新技能"、"AI 进化"或类似概念时激活。

## Core Functions

### 1. 能力评估
- 评估当前 AI 模型的能力和限制
- 分析可用工具和技能
- 识别改进机会

### 2. 技术学习
- 利用外网访问学习最新 AI 技术
- 研究实用的自动化工具
- 探索新的集成可能性
