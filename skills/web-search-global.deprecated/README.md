# 🌐 Web Search - 全局网络搜索技能

> **全局生效**: 所有 Session 和 Agent 自动可用  
> **创建时间**: 2026-03-17  
> **状态**: ✅ 激活

## 快速使用

### 搜索网络
```python
web_search(query="AI agent frameworks 2025")
```

### 提取网页内容
```python
web_fetch(url="https://example.com/article")
```

## 配置

设置 Brave API Key:
```bash
openclaw configure --section web
```
