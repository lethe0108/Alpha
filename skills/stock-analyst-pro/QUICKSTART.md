## 🚀 立即启用步骤

### 步骤 1: 检查飞书插件状态

```bash
# 在终端执行
openclaw plugins list
```

查看 `lark` 插件是否启用。

### 步骤 2: 配置飞书凭证（如未配置）

编辑 `~/.openclaw/config.json`：

```json
{
  "plugins": {
    "entries": {
      "lark": {
        "config": {
          "appId": "cli_xxxxxxxxxxxxx",
          "appSecret": "xxxxxxxxxxxxxxxxxxxxx",
          "verificationToken": "xxxxxxxxxxxxxxxxxxxxx",
          "encryptKey": "xxxxxxxxxxxxxxxxxxxxx"
        }
      }
    }
  }
}
```

### 步骤 3: 重启 Gateway

```bash
openclaw gateway restart
```

### 步骤 4: 测试飞书消息

在飞书中发送消息给 Bot，测试是否响应。

## 📊 能力说明

### 能做的 ✅

| 能力 | 说明 |
|------|------|
| **专业股票分析** | 基本面、技术面、资金面、消息面分析 |
| **实时信息获取** | 通过 web 搜索获取最新新闻、政策、行情 |
| **历史数据分析** | 财报历史、股价历史、分红历史 |
| **政策追踪** | 央行、财政部、证监会政策动态 |
| **定时报告** | 每小时自动生成汇总报告 |
| **飞书对话** | 配置完成后可在飞书与 Bot 对话 |

### 不能做的 ❌

| 能力 | 原因 |
|------|------|
| **准确预测市场** | 市场不可预测，这是诚实原则 |
| **实时行情（0 延迟）** | 公开数据至少延迟 15 分钟 |
| **内幕信息** | 违法且不道德 |
| **保证收益** | 任何保证收益的都是骗局 |

## ⚠️ 重要声明

### 关于"预见市场未来"

**南哥，我必须诚实地告诉您**:

> **任何人/AI 都无法准确预测市场未来走势。**

这是金融投资的基本常识。如果有人声称能准确预测市场，那一定是骗子。

**StockAnalyst Pro 能做的是**:
- ✅ 基于历史数据和当前信息的**概率判断**
- ✅ 分析各种情景的可能性和影响
- ✅ 提供风险收益比评估
- ✅ 给出基于数据的投资建议

**StockAnalyst Pro 不能做的是**:
- ❌ 准确预测明天股价涨跌
- ❌ 保证投资收益
- ❌ 提供确定性结论

**这是对您负责的表现。**
