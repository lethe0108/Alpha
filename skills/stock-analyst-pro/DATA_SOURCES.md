# 数据源配置

> **版本**: v2.0 (精简版)

---

## 📊 数据源优先级

### A 级（最权威）

| 数据源 | 用途 |
|--------|------|
| 巨潮资讯 | 公司公告、财报 |
| 证监会官网 | 监管信息、处罚 |
| 交易所官网 | 交易数据、公告 |

### B 级（专业财经）

| 数据源 | 用途 |
|--------|------|
| 新浪财经 | 行情、财报、新闻 |
| 东方财富 | 资金流向、估值 |
| 同花顺 | 技术指标、资金 |

## 🔍 搜索关键词模板

| 用途 | 百度搜索关键词 |
|------|--------------|
| 行情 | `{股票代码} {股票名称} 最新股价` |
| 资金 | `{股票代码} 主力资金流向` |
| 技术 | `{股票代码} 周 K 线 月 K 线` |
| 估值 | `{股票代码} PE PB 历史分位` |
| 财报 | `{股票名称} {年份} 年报/季报` |
| 政策 | `{行业名称} 十五五规划 政策` |

## ⏰ 数据时效要求

| 数据类型 | 时效要求 |
|---------|---------|
| 股价行情 | 当日 |
| 资金流向 | 当日 |
| 成交量/换手率 | 当日 |
| 估值数据 | 当日 |
| 财报数据 | 最新季报 |
| 政策新闻 | 7 日内 |
